## 🎯 What Is Loading Bharat?

**Loading Bharat** is a satirical Digital India government portal simulation that puts citizens through the most authentically painful bureaucratic onboarding experience ever designed.

To register as a citizen you must survive:

- A **12-second biometric facial scan** with chaotic zoom shifts
- A **patriotic loyalty quiz** that auto-corrects your "No" to "Yes, Always"
- A **Pothole CAPTCHA** — audio challenge available only in Sanskrit
- A **password** requiring an Emoji + Roman Numeral + Devanagari character
- A **submit button** that runs away from your cursor
- A **math-puzzle OTP** ("OTP-{answer}-BHARAT") delivered via local post office
- A **clearance form** that physically explodes across your screen
- **7 mandatory state news broadcasts** you cannot skip
- A **GitHub star + Maa Kasam oath** before accessing any government service

*Refreshing cancels everything. Going back adds fines. The chatbot analyzes your patriotism.*

```

    ├── Phase 1: 12-second biometric scan (with chaotic CSS zoom)

    ├── Backspace adds random characters instead of deleting

    ├── Blocking "URGENT NOTIFICATION" popup (3 seconds)

    ├── Must be formatted: "OTP-{answer}-BHARAT"
```

## 🚀 Running Locally

```bash
# Clone the repository
git clone https://github.com/mannatgupta146/Loading-Bharat.git
cd Loading-Bharat

# Install & run frontend
cd frontend
npm install
npm run dev

# Install & run backend (separate terminal)
cd ../backend
npm install
npm run dev
```

Frontend runs at `http://localhost:5173`
Backend runs at `http://localhost:5000`
